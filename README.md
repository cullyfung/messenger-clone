## Introduction
An easy-to-use chat application
## Stack

- Nextjs
- NextAuth
- Zustand
- Prisma
- Tailwind CSS
- react-hook-form
- axios
- typescript
- mongodb
- pusher

## Getting Started
### Install packages

```bash
pnpm i
```

### Setup env

```bash
DATABASE_URL=
NEXTAUTH_SECRET=

GITHUB_ID=
GITHUB_SECRET=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=
NEXT_PUBLIC_CLOUDINARY_CLOUD_SECRET=

NEXT_PUBLIC_PUSHER_APP_KEY=
PUSHER_APP_ID=
PUSHER_SECRET=
```
### Setup Prisma
```bash
npx prisma db push
```

### Run

```bash
pnpm dev
```
